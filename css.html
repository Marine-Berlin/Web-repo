Formation CSS

CSS permet de styliser le HTML : couleurs, espacements, polices, disposition.

1. Cibler un élément

p {
  color: #7C9ACC;
  font-size: 18px;
}

2. Créer une classe

Les classes commencent par un point.

.important {
  background-color: #D0E7DA;
  padding: 10px;
  border-radius: 8px;
}

3. Mettre en page

Utilisez Flexbox pour aligner des blocs.

.container {
  display: flex;
  gap: 16px;
}

4. Effets visuels

Ajoutez des transitions pour rendre l'interface plus fluide.

button {
  transition: transform 0.2s ease;
}
button:hover {
  transform: translateY(-2px);
}
