Bienvenue sur 3M_learning ! Cette plateforme propose une formation simple, claire et 100% gratuite pour apprendre HTML, CSS et JavaScript.

Commencer avec HTML Tester JavaScript

Pourquoi ce site ?

Le contenu est progressif pour éviter les blocages : vous apprenez les bases, vous pratiquez immédiatement, puis vous avancez vers des notions plus complètes.

Comment suivre la formation ?

  1. Commencez par la page HTML pour structurer vos pages.
  2. Passez à CSS pour améliorer le style visuel.
  3. Terminez par JavaScript pour ajouter de l'interactivité.
  4. Utilisez les zones de test pour voir les résultats en direct.

Formation accessible

Chaque leçon est expliquée en français, avec des exemples simples et des boutons de test qui ouvrent une page locale pour pratiquer.
